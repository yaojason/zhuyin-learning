# 注音學習網站設計規格

## 概要

給 6 歲幼稚園兒童的注音練習網站。小孩認得注音但會忘記，需要每天寫日記鞏固。
本機跑、純前端、單一 HTML 檔案。

## 目標使用者

- 6 歲兒童，已學過注音但需反覆練習
- 家長陪伴或兒童獨立使用

## 技術架構

- **單一 HTML 檔案**：內嵌 CSS + JS，開瀏覽器就用
- **路徑**：`/Users/jason/Claude_env/zhuyin/index.html`
- **資料存儲**：localStorage（無需後端、無需帳號）
- **TTS**：Edge TTS（免費 Azure 語音 `zh-TW-YunJheNeural`）
  - 37 個注音符號 + 常用拼音組合：預錄 mp3（離線可用）
  - 日記朗讀：即時合成（需網路）
- **語音辨識**：Web Speech API（瀏覽器內建）
- **手寫辨識**：Canvas 筆畫路徑比對（37 個注音模板）

## 頁面結構

底部 Tab 切換，大圖示：

| Tab | 圖示 | 功能 |
|-----|------|------|
| 閃卡 | 🃏 | 注音符號認讀 |
| 拼音 | 🧩 | 聲母+韻母+聲調組合 |

## 功能規格

### 1. 閃卡

**注音符號**：37 個
- 聲母 21 個：ㄅㄆㄇㄈㄉㄊㄋㄌㄍㄎㄏㄐㄑㄒㄓㄔㄕㄖㄗㄘㄙ
- 韻母 16 個：ㄚㄛㄜㄝㄞㄟㄠㄡㄢㄣㄤㄥㄦㄧㄨㄩ

**操作流程**：
1. 畫面中央大卡片顯示注音符號
2. 點卡片 → 播放發音（預錄 mp3，唸兩遍，中間停 0.5 秒）
3. 播放時符號放大 + 彈跳動畫（視覺輔助）
4. 下方兩按鈕：✅ 認得 / ❌ 不確定
5. 翻下一張

**學習模式**：
- 全部（37 個隨機）
- 聲母 / 韻母（分組）
- 弱項複習（只出 ❌ 的 + 日記寫錯的）

**間隔重複**：標記 ❌ 的符號出現頻率提高

### 2. 拼音練習

**三步拼音**：聲母 → 韻母 → 聲調

**操作流程**：
1. 出一個目標字（例如「媽」，上方標注音 ㄇㄚ）
2. 選聲母（4 個選項）
3. 選對 → 選韻母（4 個選項）
4. 選對 → 選聲調（一二三四聲 + 輕聲）
5. 三步都對 → 播放完整發音 + 星星動畫
6. 選錯 → 震動 + 正確答案亮起，再試一次

**難度遞進**：
- Level 1：常用字、2 個選項
- Level 2：4 個選項
- Level 3：不給目標字，只播放音，要拼出來

### 3. 聽音選字（拼音模式）

**操作流程**：
1. 大喇叭按鈕 🔊
2. 點喇叭 → 播放完整拼音（例如「ㄇㄚ」）
3. 下方出現 4 組注音組合讓小孩選
4. 選對 → 星星 + 下一題
5. 選錯 → 正確答案閃爍 + 再播一次音
6. 可重複點喇叭

### 4. 日記

**核心理念**：降低門檻，系統做前置作業，小孩只需認和寫。

**流程**：
```
【系統做的（前置）】
1. 出圖片提示引導（今天吃了什麼？看到什麼？做了什麼？）
2. 小孩按住麥克風說話 → 語音辨識轉注音

【小孩做的】
3. 看注音 → 認讀（常忘的字標紅色虛線）
4. 手寫/描寫那些字（有筆順動畫提示）
5. 寫完 → 系統唸回來 → 成就感
```

**手寫模式**：
- Canvas 畫布，用手指/滑鼠寫
- 筆畫路徑比對 37 個注音模板
- 辨識對就留下，不對可重寫
- 提供筆順動畫引導（不是考試是教學）

**語音模式**：
- 按住 🎤 說話
- Web Speech API 辨識 → 轉注音顯示
- 辨識失敗：「再說一次試試看 🎤」（正面語氣）

**弱項連動**：
- 日記中寫錯/猶豫的注音 → 自動加入閃卡弱項
- 隔天閃卡優先出這些字

**儲存**：每天一篇，localStorage

### 5. 家長後台

**解鎖**：長按右上角齒輪 3 秒

**內容**：
- 今日統計：練習分鐘、題數、正確率
- 7 日趨勢：柱狀圖
- 弱項清單：錯最多的 Top 5 注音
- 日記回顧：按日期瀏覽
- 設定：
  - 每日目標題數（預設 20）
  - 發音速度（慢 0.6 / 正常 1.0）
  - TTS 聲音選擇（2-3 種）
  - 重置進度

### 6. 獎勵機制

- 每答對 5 題得 ⭐
- 首頁顯示累積星星（集點卡）
- 連續天數 🔥（連續 3 天！）
- 日記累積變成「繪本」可翻閱

## TTS 設定

**語音**：`zh-TW-YunJheNeural`（雲哲，少年聲，活潑）
**語速**：預設 rate 0.6（慢而清晰）
**重複**：每個音唸兩遍，中間停 0.5 秒
**備選聲音**：`zh-TW-HsiaoChenNeural`（曉臻，女聲）

**預錄策略**：
- 37 個注音符號單音 → mp3
- ~200 個常用拼音組合 → mp3
- 離線時用預錄，線上時可即時合成

## UI 設計原則

**小孩模式**：
- 超大按鈕（最小 60px）
- 可愛圓角卡片
- 明亮配色（不刺眼）
- 全注音/圖示引導，零文字選單
- 操作反饋：彈跳、星星、音效

**回應語氣**：
- 正確：「好棒！」「太厲害了！」⭐
- 錯誤：「再試一次！」「沒關係，看看這個」（絕不說「錯了」）
- 鼓勵為主，永遠正面

## 不做的事（YAGNI）

- 不做多人/帳號系統
- 不做後端 server
- 不做 ML 手寫辨識（用筆畫比對就夠）
- 不做自訓 TTS 模型
- 不做 app（先做網頁）
- 不做國字教學（只教注音）
